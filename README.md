# PharmaPulse Analytics Dashboard

Interactive Power BI dashboard for pharmacy sales, profitability, product performance, regional insights, and trend analysis.

## Project Overview

This project is an interactive Power BI dashboard designed to analyze pharmacy sales performance, profitability, product trends, regional insights, and promotional impact.

The dashboard provides business intelligence through KPI cards, trend analysis, product performance tracking, geographic analysis, and profitability insights using Power BI, Power Query, DAX, and data modeling techniques.

## Dashboard Pages

1. Executive Overview
2. Product Analysis
3. Pharmacy & Regional Insights
4. Time & Trend Analysis
5. Profitability & Promotions

## Key Features

* KPI Cards for Revenue, Profit, Orders, and Margin
* Monthly and Quarterly Revenue Analysis
* Product Performance Analysis
* Generic vs Non-Generic Product Insights
* Promotion Impact Analysis
* Regional and Country-wise Sales Analysis
* Geographic Map Visualizations
* Profitability and Margin Tracking
* Interactive Filters and Slicers
